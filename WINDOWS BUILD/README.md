# WINDOWS BUILD

## Download and run (staff)
Pre-built Windows app zip in this repo:

- `WINDOWS BUILD/dist/win_SCU_CBoards.zip`

Steps:
1. Download that zip from GitHub.
2. Extract all files.
3. Open the extracted `win_SCU_CBoards` folder.
4. Double-click `win_SCU_CBoards.exe`.

Do not run `build_windows.bat` unless you are creating a new build.

## Build from source (developers)
Source and build script live in:

- `WINDOWS BUILD/Windows_Build_Source/`

Copy that folder to a Windows PC, then double-click `build_windows.bat`.
